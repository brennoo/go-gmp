# GO-GMP

### WIP fork that is updating the pkg to support recent GMP protocol

Library to interact with the Greenbone Vulnerability Manager using the [gmp protocol](https://docs.greenbone.net/API/GMP/gmp-22.6.html) (Greenbone Management Protocol, `version 22.6`)

Here you willl find methods to create tasks, targets, scanners and retrieve results generated by the Openvas scanner.

* This library is compatible with the latest version of GVM: 22.6
